- Chạy client
```bash
cd ./client
npm install
npm run dev
```
Client chạy ở port 3000: http://localhost:3000

- Sử dụng file config.js ở trong drive thay cho file config ở server (tránh lộ key send email)
- Chạy server
```bash
cd ./server
npm install
npm start
```
Server chạy ở port 8000
